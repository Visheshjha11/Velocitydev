## What is Velocity.dev?

Velocity.dev is a modern code typing platform built for developers who want to train on actual programming syntax instead of generic text passages.

The platform combines a fast, minimal interface with syntax-aware typing exercises, live performance analytics, configurable challenges, and smooth interactions designed to feel like a professional developer tool rather than a traditional typing website.

From frontend engineers to competitive programmers, Velocity.dev provides a focused environment for building typing speed directly on real code.

---

## Core Features

- Real-world code typing challenges
- Syntax-highlighted typing interface
- Live WPM, CPM, and accuracy metrics
- Adjustable duration and difficulty settings
- Blind mode support
- Auto-indentation handling
- Smooth motion system powered by Framer Motion
- Responsive experience across desktop and mobile
- Dark and light themes
- Fast Vite-powered architecture
- Minimal, distraction-free UI system

## Local Development

### Clone Repository

```bash
git clone https://github.com/Visheshjha11/velocitydev.git
```

### Navigate Into Project

```bash
cd velocitydev
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

Application runs locally at:

```bash
http://localhost:5173
```
